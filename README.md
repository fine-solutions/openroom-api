# OpenRoom

OpenRoom - это система контроля и управления доступом к помещениям организации. Предназначена, прежде всего, для планирования лекций и внеучебных мероприятий в аудиториях ВУЗа.


## Запуск

С помощью Docker Compose:
``` shell
docker compose up
```
